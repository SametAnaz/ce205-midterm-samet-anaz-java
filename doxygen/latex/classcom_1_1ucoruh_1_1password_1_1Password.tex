\doxysubsection{Detailed Description}
Represents a stored password entry for a specific service. 

\begin{DoxyAuthor}{Author}
Password Manager Team 
\end{DoxyAuthor}
\begin{DoxyVersion}{Version}
1.\+0
\end{DoxyVersion}
This class provides a structure to store and manage credentials such as service name, username, and password. You can retrieve or update these fields through the provided getter and setter methods.

Additionally, this class maintains a history of previous passwords using a doubly linked list data structure for efficient insertion and traversal.
